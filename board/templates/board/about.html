My Web – 개인 포트폴리오 프로젝트

백엔드 학습의 결과물을 축적하고, 글을 쓰고, 프로젝트를 정리하기 위해 만든 개인 사이트입니다.

개요

이 프로젝트는 Django 기반으로 제작된 블로그/포트폴리오 웹사이트입니다. 기록 중심의 플랫폼을 스스로 구축함으로써, 프레임워크의 전반적인 동작 원리를 이해하고 CRUD, 인증, ORM 등 웹 개발의 필수 요소들을 실제로 경험하는 것을 목표로 했습니다.

기술 스택

  • Django, Python
  • HTML/CSS (Vanilla), Django Templates
  • SQLite → (향후 PostgreSQL 전환 예정)
  • Gunicorn + Nginx 배포 예정

주요 기능

  • 게시글 작성, 수정, 삭제 (CRUD)
  • 사용자 인증 기반 게시글 관리
  • 게시글 리스트/상세 페이지
  • About(포트폴리오) 페이지

프로젝트 구조도
