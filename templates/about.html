My Web – 개인 블로그 프로젝트

2025 11월부터 서비스해오고 있는 개인 사이트입니다. 백엔드 학습의 결과물을 축적하고, 글을 쓰고, 프로젝트를 정리하기 위해 만들었습니다.

개요

이 프로젝트는 Django 기반으로 제작된 블로그/포트폴리오 웹사이트입니다. 기록 중심의 플랫폼을 스스로 구축함으로써, 프레임워크의 전반적인 동작 원리를 이해하고 CRUD, JWT 인증, ORM 등 웹 개발의 필수 요소들을 실제로 경험하는 것을 목표로 했습니다.

기술 스택

  • 백엔드: Django (Python), Django REST Framework (DRF)
  • 데이터베이스: PostgreSQL
  • 인증/인가: JWT
  • 프론트: HTML/CSS, Django Templates
  • 배포 (Deployment): Docker, AWS EC2
  • 웹 서버: Gunicorn (WSGI) + Nginx (Reverse Proxy)

주요 기능

  • 게시글 작성, 수정, 삭제 (CRUD)
  • 사용자 인증 기반 게시글 관리
  • 게시글 리스트/상세 페이지
  • About(포트폴리오) 페이지

프로젝트 구조도

my-web/
 ├─ accounts/    (User 인증/회원가입 담당)
 ├─ posts/       (Post 모델, Views 담당)
 ├─ comments/    (Comment 모델, Views 담당)
 │
 ├─ config/      (settings, urls, wsgi)
 ├─ templates/   (base.html, about.html 등 공통 템플릿)
 ├─ static/      (공통 CSS/JS)
 ├─ docker-compose.yml
 └─ Dockerfile

개선 예정

  • DRF를 활용한 댓글 API 전환 (재귀적 Serializer 구현 예정)
  • 프로젝트 목록 페이지 추가
  • 검색 기능 추가
  • 카테고리 기능 추가
